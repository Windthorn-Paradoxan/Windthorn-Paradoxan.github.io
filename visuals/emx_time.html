1. Classical Time: One Thin Line

Linear Time Model

In the classical picture, time is a single line. The universe has one history; you move along it like a bead on a wire.

Any attempt to “go back” to an earlier point is assumed to change that same line. That’s where paradoxes like “What if I prevent my grandfather’s birth?” come from: you’re trying to overwrite a fixed, single history.

In EMx terms, this is a collapsed projection: only one branch of the full lattice is being viewed.

Good for simple causality. Bad for self-consistent loops.

2. EMx View: Time as a Lattice of States

State Lattice, Not a Single Track

EMx treats “time” as a sequence of visited states in a larger lattice of possibilities. Each node is a full configuration of the universe; each edge is a legal transition that respects polarity, conservation, and no-clone constraints.

What we call “the timeline” is really just one path through this lattice. If a process ever “returns” to an earlier configuration, it doesn’t overwrite the old path — it creates a new branch that happens to pass through a similar state.

The past remains fixed as one already-traversed path. New excursions produce additional branches, not contradictions.

3. “Grandfather Paradox” Under EMx

Why the Paradox Vanishes

In EMx, a “time jump” is just another state transition in the lattice, obeying the same polarity / conservation / no-clone rules as everything else.

If an advanced process creates a path that loops back near an earlier configuration (your grandfather’s era), it doesn’t overwrite the existing branch. It creates a new path that also passes through a compatible state.

The earlier branch — the one where you were born — still exists as a completed path. The new branch may arrive in a similar macro-state (same grandfather, same year), but it’s a different node in the lattice: subtly different boundary conditions, different observer history, different phase index.

You can interfere with this branch’s “grandfather”, but that does not retroactively remove the branch that already contained you. The system never needs to solve a logical contradiction — it simply maintains both paths in parallel.

4. EMx Summary of Time

In short, EMx treats time as “loops in loops” of state transitions across a structured lattice, not a single editable tape. “Travel” is just a special kind of transition.

The rules are simple:

  • The past path is fixed as a completed traversal.
  • New transitions can re-enter similar regions of the lattice but cannot erase or overwrite an already realized branch.
  • No-clone constraints prevent contradictions like “I both exist and never existed” on the same branch.
  • What we call “paradox” is really just trying to force a single-line picture onto a multi-branch structure.

Seen this way, “time travel” is not a magical exception to physics — it’s just one more way to move through a space of states, subject to the same balance and conservation rules as everything else.
